# GTI525 EQUIPE 10

## Setting up and running dev server
delete `node_modules` folder if exists

Then install dependencies
```bash
npm install
```

then you can run server with
```bash
npm run dev
```

## Scripts 

To build
```bash
npm run build
```

To check lint 
```bash
npm run lint
```

To Fix linting errors/warnings
```bash
npm run lint-fix
```
